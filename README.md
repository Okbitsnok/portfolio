# Dmitry Kontsybko — Frontend Developer Portfolio

![Portfolio Preview](preview.png)

🌐 **Live demo:** https://kontsybko-web-portfolio.netlify.app/

A modern, animated portfolio website built with pure HTML, CSS, and JavaScript.

## ✨ Features

- Custom animated cursor with lag effect
- Particle system with canvas
- Scroll reveal animations
- Floating code card with typing cursor
- Responsive design
- Pure HTML/CSS/JS — no frameworks

## 🗂 Structure

```
portfolio/
├── index.html          # Main HTML
├── css/
│   ├── variables.css   # CSS variables & design tokens
│   ├── base.css        # Reset, cursor, noise overlay
│   ├── nav.css         # Navigation
│   ├── hero.css        # Hero section
│   ├── sections.css    # About, Work, Services, Contact, Footer
│   └── animations.css  # Scroll reveal & utility animations
└── js/
    ├── cursor.js       # Custom cursor logic
    ├── particles.js    # Canvas particle system
    └── main.js         # Scroll reveal & skill bars
```

## 🚀 Run locally

Just open `index.html` in your browser — no build step needed!

## 📦 Tech Stack

HTML · CSS · Vanilla JS · Canvas API · Google Fonts
